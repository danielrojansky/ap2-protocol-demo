AP2 Protocol API Documentation

Agent Endpoints

POST /api/agents/shopping/intent

Create an intent mandate from natural language description

Example Request 
{
  "description": "I want to buy wireless headphones under $100",
  "max_amount": 100,
  "categories": ["electronics"],
  "user_id": "user_123"
}
POST /api/agents/merchant/search

Search products based on intent mandate

Example Request 
{
  "intent_mandate_id": "intent_123",
  "search_terms": ["wireless", "headphones"],
  "max_price": 100
}
POST /api/agents/payment/process

Process payment with mandate authorization

Example Request 
{
  "payment_mandate_id": "payment_123",
  "otp_code": "123456",
  "risk_data": {...}
}
